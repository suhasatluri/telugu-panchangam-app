## How to be added

When your first pull request is merged, open a follow-up PR adding your
name (or open an issue and we'll do it). Use this format:

```markdown
- **Your Name** ([@github-handle](https://github.com/github-handle)) — City, Country
  One line about what you contributed
```

You can also list:
- A link to your website if you'd like attribution there
- The language(s) you can help with (Telugu, Hindi, English, etc.)

If you'd prefer to remain anonymous, that's completely fine too — just say
so in your PR and we'll respect it.

## Other ways to recognise contributors

- Significant contributions are highlighted in [CHANGELOG.md](CHANGELOG.md)
  release notes
- Festival accuracy corrections are credited inline next to the date in
  the engine source where appropriate
- The GitHub "Contributors" graph at
  https://github.com/suhasatluri/telugu-panchangam-app/graphs/contributors
  is updated automatically
